# Job Market Analytics

## Overview
This project analyzes job market trends using Python.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Salary Trend Analysis
- Skill Demand Analysis
- Visualization using Matplotlib/Seaborn

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Outcome
Identified high-demand skills and job trends from datasets.

## How to Run
1. Install dependencies
2. Run analysis.py
